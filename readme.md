# 아파트 관리 프로그램
* 아파트 센서를 기반으로 방의 온도, 습도, 전력량을 알려준다.
* 실시간 그래프 제공
* 일, 월, 달별 평균 제공

# 설치
* npm i

# 실행
* > npm start
* > npm test

# dotenv

KAKAO_ID=example
